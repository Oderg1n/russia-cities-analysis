# russia-cities-analysis
Анализ датасета городов России

Исследовательский проект по анализу географических, демографических и исторических характеристик городов Российской Федерации

Цель проекта — провести комплексный исследовательский анализ датасета городов России, включающий:
- Изучение распределения городов по регионам и федеральным округам
- Анализ демографических показателей (население)
- Исследование исторической динамики (годы основания)
- Геопространственный анализ (координаты, климат)
- Выявление взаимосвязей между переменными
- Обогащение данных внешними источниками (климат)

Основной файл: city.csv (~1117 городов, 25+ переменных)






Данные из открытых источников РФ. Аналитический код распространяется под лицензией MIT.
